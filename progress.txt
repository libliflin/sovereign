## Codebase Patterns
(Updated by Ralph after each iteration — check here first for conventions discovered)

- Domain variable: always {{ .Values.global.domain }} in Helm templates
- Storage class: always {{ .Values.global.storageClass }} — default is "ceph-block"
- All ingresses use TLS, cert from cert-manager
- Keycloak OIDC client IDs follow pattern: sovereign-<service>
- ArgoCD apps live in argocd-apps/<tier>/<service>-app.yaml
- Namespace naming: one namespace per service, named after the service
- SSH_OPTS in shell scripts must be a bash array (not a string) to satisfy shellcheck SC2086 — use `SSH_OPTS=(-o ... -i "$KEY")` and expand as `"${SSH_OPTS[@]}"`
- `grep -v 'x' | wc -l` triggers SC2126 — use `grep -vc 'x' || true` instead (the `|| true` guards against grep returning exit 1 when there are 0 matches, which would trip `set -e`)
- New branch for each story must be created from main and then merge in prior story branches to build on scaffold work
- prd.json only exists at root (not scripts/ralph/) — only update the root copy
